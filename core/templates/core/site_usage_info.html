{% trans 'QA-SITE INFO' %}

{% blocktrans %} This section concerns the questions part of the website (academic and discussion questions). {% endblocktrans %}

{% trans 'Questions and Answers' %}
{% blocktrans %} By post, we refer to a question or answer.
You get {{ ask_qstn_points }} points when you ask a question. When your post is liked(upvoted), you get {{ upvote_points }} points. You are deducted {{ downvote_points }} points when your post is downvoted(disliked).
Anyone can upvote a post, but to downvote a post, the user needs to have {{ downvote_req_points }}; this is to ensure a post can be downvoted only by users that have used the website for some time.
You get {{ aca_ans_points }} points when you answer an academic question and {{ school_ans_points }} when you answer a school question. You are not rewarded any point for answering your own questions. However you will be rewarded points when these answers are upvoted. Your answers should be valuable else these points will not be worth much. You may have at most {{ max_answers }} answers per question. {% endblocktrans %}
{% trans 'Comments' %}
{% blocktrans %} Comments are used to ask for clarification and extra info concerning a question or answer, or to add information you think others may find useful. You get no points when you comment on posts. However, the number of upvotes on your comments means a lot. So ensure to contribute positively to questions and answers. {% endblocktrans %}
{% trans 'Editing' %}
  • {% blocktrans %} Questions: Only the poster of a question can edit it. However, questions with more than {{ cannot_edit_qstn_vote }} votes (upvotes - downvotes) or more than {{ cannot_edit_qstn_ans_count }} answer can't be edited. {% endblocktrans %}
  • {% blocktrans %} Answers: Only the poster of an answer can edit it. However, answers with more than {{ cannot_edit_ans_vote }} votes can't be edited. {% endblocktrans %}
  • {% blocktrans %} Comments: Only the poster of a comment can edit it. However, comments with more than {{ cannot_edit_comment_vote }} votes can't be edited. {% endblocktrans %}
{% trans 'Deleting' %}
  • {% blocktrans %} Questions: Questions with more than {{ cannot_del_qstn_vote }} votes or more than {{ cannot_del_qstn_ans_count }} answer cannot be deleted by the poster. This is because we consider such questions to be valuable and they may help future users that share similar questions. {% endblocktrans %}
  • {% blocktrans %} Answers: Answers with more than {{ cannot_del_ans_vote }} votes can't be deleted for the same reason as above. {% endblocktrans %}
  • {% blocktrans %} Comments: Comments with more than {{ cannot_del_comment_vote }} votes can't be deleted because they may contain information valuable to future users. {% endblocktrans %}
